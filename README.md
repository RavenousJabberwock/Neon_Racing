# 🚘 Neon Racing

**Neon Racing** is a fast-paced, retro-futuristic arcade racing game. Experience vibrant neon aesthetics, challenging tracks, and responsive controls built straight for the browser.

### 🎮 [Play Neon Racing Live on GitHub Pages]([https://github.io](https://ravenousjabberwock.github.io/Neon_Racing/))
* https://ravenousjabberwock.github.io/Neon_Racing/

---

## 🚀 Features

* **Vibrant Visuals:** Stunning cyberpunk and synthwave-inspired neon graphics.
* **Responsive Handling:** Smooth keyboard controls optimized for desktop browsers.
* **Instant Play:** No installations required—loads directly inside your web browser.
* **Performance Tracking:** High-score mechanics or lap-timers to test your skills.

## 🕹️ How to Play

### Controls
* **Accelerate:** `W` or `Up Arrow`
* **Brake / Reverse:** `S` or `Down Arrow`
* **Steer Left:** `A` or `Left Arrow`
* **Steer Right:** `D` or `Right Arrow`

### Objective
Navigate down the neon road, dodge other cars, collect bonuses, maximize your speed, and beat your high score!

## 🛠️ Built With

* **HTML5 & CSS3:** For the game structure and styling layout.
* **JavaScript (ES6):** Powering the core game loop, physics engine, and controls.
* **HTML5 Canvas API:** Handling high-performance 2D/3D rendering.

## 📦 Running Locally

If you want to modify the code or run the project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate into the directory:**
   ```bash
   cd Neon_Racing
   ```

3. **Launch a local server:**
   Because web browsers heavily restrict local file access (`file://`), use a simple server.
   * If you have Python installed:
     ```bash
     python -m http.server 8000
     ```
   * If you use VS Code, install the **Live Server** extension and click **Go Live**.

4. **Open in browser:**
   Navigate to `http://localhost:8000` (or your local port) to test changes dynamically.
